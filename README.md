# Fluxus Digital

Designed and developed an offline financial management software focused on privacy, local cash flow control, and cost reduction for end users (B2C).

## 🚀 Key Features
* **Business Logic:** Implemented entirely in Python for efficient data processing.
* **Data Security:** Robust local security systems, including encryption and key generation.
* **Product Lifecycle:** Full management from development and direct distribution to tech support and user-feedback bug resolution.
